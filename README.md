# Sen Physiotherapie – Website

Website for a physiotherapy practice, developed as a freelance project.

## Tech Stack
- React 18
- React Router
- Framer Motion (animations)
- Font Awesome / Heroicons
- Netlify

## Features
- Responsive design
- Service overview
- Gallery
- Contact form with reCAPTCHA
